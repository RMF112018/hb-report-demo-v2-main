# Project Page Overview

## One Workspace for Everything

This central hub organizes every project activity—from preconstruction to closeout—into an elegant, modular workspace. The Project Page is designed to offer total visibility, with structured views for financials, procurement, staffing, and field performance—all customized per role.

Whether you're a Project Manager, Executive, Estimator, or Coordinator, this page connects every team and tool.

### Key Benefits

- **Unifies all project tools and insights** into one location
- **Enables seamless transitions** between phases of work
- **Context-aware tabs** provide the right features to the right people
- **Optimized for speed, visibility, and decision-making** with HBI
